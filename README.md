# XAILON - Desarrollo Web de Élite

Sitio web profesional para Xailon Digital Studio, una agencia de desarrollo web especializada en crear experiencias digitales de alto impacto.

## Características
- Diseño moderno y minimalista
- Totalmente responsivo (móvil, tablet, desktop)
- Menú hamburguesa para móviles
- Formulario de contacto
- Integración WhatsApp y Email
- Newsletter

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Secciones
- Inicio
- Servicios (E-commerce, Webs Corporativas, Sistemas Pro)
- Portfolio
- Contacto

## Cómo usar
1. Abre `index.html` en tu navegador
2. Personaliza los textos en el archivo HTML
3. Cambia los colores en `style.css`
